# Team Task Manager

A modern full-stack MERN application for managing projects, assigning tasks, and tracking team progress with role-based access control.

---

# Features

## Authentication
- User Registration
- User Login
- JWT Authentication
- Protected Routes

## Role-Based Access
### Admin
- Create Projects
- Create Tasks
- Edit/Delete Tasks
- View All Projects & Tasks

### Member
- View Assigned Tasks
- Update Task Status
- View Assigned Projects

## Task Management
- Create Tasks
- Assign Members
- Task Status Tracking
- Priority System
- Edit/Delete Tasks
- Search Tasks

## Dashboard
- Analytics Charts
- Task Statistics
- Recent Tasks
- Project Overview

## UI/UX
- Modern SaaS UI
- Responsive Design
- Loading States
- Priority Badges
- Clean Dashboard Layout

# Installation

## Clone Repository

```bash
git clone https://github.com/sandeshtiwari2211047/Team-Task-Manager.git
```

---

# Backend Setup

```bash
cd backend
npm install
npm start
```

Create `.env` file inside backend:

```env
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret_key
```

---

# Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Create `.env` file inside frontend:

```env
VITE_API_URL=http://localhost:5000/api
```
